# Contributing

This is a personal automation project and is **not accepting contributions** at this time.

If you find this project useful and want to use it for your own purposes, feel free to fork it.

## Issues

If you encounter a bug or have a question, you can open an issue, but please understand that responses may be delayed or not provided.

## Forking

You're welcome to fork this repository and modify it for your own use. The code is provided as-is.
